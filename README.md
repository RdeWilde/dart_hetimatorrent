# dart_hetimatorrent

A Torrent library for Dart developers. 

- tracker server library
  - http only (Alpha level)
- tracker client library
  - http only (Alpha level)
- torrent client library
  - now creating (bep3 pre-Alpha level)
- dht tracker
  - now creating (bep5 pre-Alpha level)
